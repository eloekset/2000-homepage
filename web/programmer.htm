Spill og programmer

 

Alle spill og programmer på denne siden er laget av meg. De er alle skrevet i Visual Basic og laget for Windows 95/98. Du må derfor ha Visual Basic 5.0 biblioteket installert på systemmappen til Windows. Dersom du har dette installert, slipper du å laste ned installasjonsprogrammene og sparer dermed mye nedlasting. VB 5.0 biblioteket finner du på microsoft's hjemmeside: ftp://ftp.microsoft.com/softlib/MSLFILES/MSVBVM50.EXE

Du kan også laste ned installasjonsprogrammet for Idioten (kabal, også kjent som "4 rekker"), som også installerer systemfilene ovenfor.
